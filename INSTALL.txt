Поправка за бутона „Изтегли комплекта“ — M5-001

Причина:
Файлът downloads/lesson-pack.zip не е бил публикуван в GitHub Pages.

Инсталиране:
1. Разархивирайте този пакет.
2. Копирайте папката resources в основната папка EM-Learning-Studio.
3. Потвърдете „Замени файловете“.
4. Направете Commit и Push origin в GitHub Desktop.
5. Отворете урока с Ctrl + F5.

След публикуване бутонът „Изтегли комплекта“ ще сваля ZIP архив,
съдържащ PDF презентациите, теста, ключа, методиката, изображенията и видеото.
