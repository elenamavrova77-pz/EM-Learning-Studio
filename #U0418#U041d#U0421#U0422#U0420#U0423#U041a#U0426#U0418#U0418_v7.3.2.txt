EM Learning Studio v7.3.2

1. Копирайте цялото съдържание на тази папка в локалната папка на GitHub проекта.
2. Изберете замяна на съществуващите файлове.
3. В GitHub Desktop направете Commit с текст:
   EM Learning Studio v7.3.2 homepage
4. Натиснете Push origin.
5. Изчакайте 1–2 минути и отворете:
   https://elenamavrova77-pz.github.io/EM-Learning-Studio/
6. Натиснете Ctrl + F5 за принудително презареждане.

Важно: качете съдържанието на папката, а не самата папка като допълнително ниво.
