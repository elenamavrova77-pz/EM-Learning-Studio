# EM Learning Studio v7.2

## Ново
- Показват се само наличните категории от `pack.json`.
- Автоматични броячи за наличните материали.
- Раздел „Методически материали“:
  - План на урока
  - Бележки на учителя
  - Оценяване
- Вграден преглед на PDF, видео, аудио и изображения.
- Галерия с увеличаване на изображенията.
- Режим „Учител“ и „Ученик“.
- Старият адрес `pack.html?id=...` автоматично пренасочва към `lesson.html?id=...`.

## Важно
Статичен сайт в Netlify не може сам да изброява файловете в папките.
Всеки реален файл трябва да бъде записан в `pack.json`.

Провери точните имена на двете презентации в:
`resources/mathematics/M6-001/pack.json`
