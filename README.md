# EM Learning Studio Portal v4

Завършена многостранична версия на платформата.

## Страници
- Начало
- Библиотека
- Математика
- Информационни технологии
- БДП
- Интердисциплинарни уроци
- Авторски проекти
- За автора
- Контакти

## Добавяне на нова игра
1. Качете HTML файла в подходяща папка в `games/`.
2. Добавете запис в `data/catalog.json`.
3. Качете промените в GitHub.
4. Netlify ще публикува автоматично.
