# EM Learning Studio v6

## Основни промени
- Единен каталог `data/resources.json`
- Универсална страница `pages/pack.html?id=...`
- Автоматична библиотека
- Търсене и филтри по направление и клас
- Препоръчани комплекти на началната страница
- Четири мигрирани учебни комплекта
- Запазени работещи HTML игри

## Добавяне на нов ресурс
1. Добавете файловете на ресурса в подходяща папка.
2. Добавете нов запис в `data/resources.json`.
3. Ресурсът автоматично се появява в библиотеката.
