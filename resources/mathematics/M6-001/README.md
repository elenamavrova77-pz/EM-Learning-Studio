# M6-001 – Задачи от движение

## Как да свържеш реалните файлове

1. Постави файловете в правилните подпапки.
2. Отвори `pack.json`.
3. В полето `file` напиши точното име и подпапка.

Примери:

```json
"file": "presentations/01-urok.pdf"
```

```json
"file": "images/infografika.png"
```

```json
"file": "audio/muzika.mp3"
```

```json
"file": "video/video.mp4"
```

За корицата:

```json
"cover": "images/cover.webp"
```

При липсващ материал остави:

```json
"file": ""
```
